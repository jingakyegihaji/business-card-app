필드 카탈로그 (전체 후보군)

- 여기서 “가능한 입력 항목”을 등록합니다. (예: mobile, fax, office_phone)
- 실제로 사용자 화면에 보일지는 템플릿별로 선택(enabledFields)됩니다.
Key / Label / Type / Required / (X)

템플릿별 입력 항목 선택 + 위치 편집

이 템플릿에서 사용할 입력 항목
- 체크된 항목만 사용자 화면에 입력 박스로 보여요.
- 체크하면 캔버스에 기본 위치가 생성되고, 드래그로 이동 가능.
- 박스를 클릭하면 선택되고, 드래그하면 위치가 바뀝니다.
